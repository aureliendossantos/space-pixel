## Télécharger sur votre ordinateur

Télécharger PICO-8 est **payant**. Si vous le possédez, **connectez-vous à votre compte** sur le site officiel de PICO-8 et **utilisez le bouton rose** pour télécharger PICO-8.

<div style="display: flex; flex-wrap: wrap; gap: 0.5rem;">
  <Button href="https://www.lexaloffle.com/pico-8.php">Site officiel de PICO-8 →</Button>
</div>

<PackageManagerTabs>
  <Fragment slot="Windows">
  1. Téléchargez le fichier pour Windows.
  2. Lancez le programme d'installation.
  3. Ouvrez l'application « PICO-8 ».
  </Fragment>
  <Fragment slot="Mac">
  1. Téléchargez le fichier pour Mac.
  2. Déplacez le fichier « PICO-8.app » dans votre dossier « Applications ».
  3. Rendez-vous dans votre dossier « Applications » et faites un clic droit sur PICO-8. Le menu déroulant s'affiche. Maintenez la touche <kbd>⌥ Option</kbd> pendant que vous cliquez sur « Ouvrir ».
  4. Confirmez que vous souhaitez ouvrir l'application.
  5. L'application est validée ! Désormais, vous pourrez ouvrir PICO-8 en cliquant sur son icône normalement.
  </Fragment>
  <Fragment slot="Linux">
  *Indéterminé.*
  </Fragment>
</PackageManagerTabs>

## Utiliser la version navigateur

Le téléchargement de PICO-8 est payant, mais **la version sur navigateur est gratuite** : c'est l'**Education Edition**, disponible à l’adresse [pico-8-edu.com](https://pico-8-edu.com).

<div style="display: flex; flex-wrap: wrap; gap: 0.5rem;">
  <Button href="https://www.pico-8-edu.com/">Utiliser PICO-8 sur le navigateur →</Button>
</div>

La version navigateur fonctionne ainsi :

- Tous les outils de création sont les mêmes.
- Certains raccourcis clavier de PICO-8 ne fonctionnent pas. Par exemple, au lieu d'utiliser <kbd>F7</kbd> pour capturer une image, il faut obligatoirement utiliser <kbd>Ctrl</kbd> <kbd>7</kbd>.
- Tu peux sauvegarder ton jeu dans un fichier à stocker sur ton ordinateur (au format `.p8` ou `.p8.png`).
- Tu ne peux pas exporter ton jeu.

:::note[Exporter, késako ?]
Exporter son jeu, c'est le transformer en une application qu'on peut lancer sur n'importe quel ordinateur, même si on n'a pas PICO-8.
:::

Pour exporter le jeu, il faut obligatoirement passer par la version téléchargeable de PICO-8, qui est payante.
