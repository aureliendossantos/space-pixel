---
title: L'éditeur de sprite
---

![](/images/memo/sprites.png)

<MemoList>

1. Les outils sont :
    - le Crayon,
    - le Tampon pour recopier le contenu du **presse-papier**,
    - la Sélection pour dessiner une sélection carrée. Vous pouvez copier la sélection dans le **presse-papier** avec <kbd>Ctrl</kbd> <kbd>C</kbd>.
    - le Décalage,
    - le Remplissage,
    - le Cercle.
2. Les *flags* (en français : « drapeaux » ou « étiquettes ») peuvent être activés sur chaque sprite. Par défaut, ils n'ont aucun effet, mais ils peuvent vous servir dans votre code, pour créer des conditions telles que « est-ce que le sprite actuel possède le flag 2 d'activé ? »
3. Le numéro du sprite, utilisé pour l'afficher avec la fonction `spr()`.
4. Les 4 onglets de la *spritesheet* (feuille de sprites). Attention : les onglets 2 et 3 partagent le même espace mémoire que la moitié basse de la carte. Dessiner dans ces onglets peut avoir des répercussions sur la carte !
5. La *spritesheet*. Le sprite 0, en haut à gauche, est un sprite spécial qui ne doit pas être modifié.

</MemoList>

:::tip[Utiliser la pipette]
En utilisant l'outil Crayon, vous pouvez récupérer une couleur du sprite avec le <kbd>Clic droit</kbd>.
:::
