<Lede> Dans ce tutoriel, vous découvrirez les fonctionnalités principales de PICO-8 en construisant un petit jeu à partir de zéro. </Lede>

<PreCheck>
  - Dessiner un personnage et le déplacer dans le jeu
  - Dessiner la carte du jeu et l'afficher
  - Créer des effets sonores
</PreCheck>

Vous voulez tester un aperçu de ce que vous allez créer ? Lancez cette démo et jouez avec les touches du clavier.

:::note[Touches du jeu]
<kbd>←</kbd> <kbd>→</kbd> <kbd>↑</kbd> <kbd>↓</kbd> Déplacer le personnage

<kbd>X</kbd> Jouer un effet sonore
:::
