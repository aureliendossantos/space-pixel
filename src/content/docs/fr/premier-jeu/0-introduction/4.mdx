:::caution[Sauvegarder, c'est la santé]
Pensez à **sauvegarder régulièrement** votre jeu avec <kbd>Ctrl</kbd> <kbd>S</kbd>.

Si vous avez fermé PICO-8 sans avoir sauvegardé, vous pouvez retrouver une **sauvegarde de secours** du jeu dans le dossier `pico-8/backup`. Demandez à votre enseignant pour plus d'informations.
:::

Lorsque vous redémarrez PICO-8, vous pouvez retrouver votre jeu en tapant :

```shell
ls
```

Puis appuyez sur <kbd>Entrée</kbd>. Cela va afficher tous les jeux existants. Pour charger celui de votre choix, utilisez :

```shell
load nom-du-jeu
```
