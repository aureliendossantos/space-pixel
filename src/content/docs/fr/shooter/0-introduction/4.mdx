  ### Teste tes connaissances

  1. Quelle est la formule à utiliser pour parcourir une séquence de données dans une boucle for en Lua ?

    <MultipleChoice>
      <Option message="Ce n’est pas la bonne réponse. La formule à utiliser est 'for i in all(tableau) do'.">for i = 1, 10 do</Option>
      <Option isCorrect message="C’est la bonne réponse. La formule à utiliser est 'for i in all(tableau) do'">for i in all(tableau) do</Option>
      <Option message="Ce n’est pas la bonne réponse. La formule à utiliser est 'for i in all(tableau) do'">for i = 1, 10, 2 do</Option>
    </MultipleChoice>

  2. Le plus souvent, comment s'appelle la variable utilisée comme compteur dans une boucle for ?

    <MultipleChoice>
      <Option message="Ce n’est pas la bonne réponse. “i” est la variable utilisée par convention, elle est dérivée du mot index.">x</Option>
      <Option isCorrect message="C’est la bonne réponse. “i” est la variable utilisée par convention, elle est dérivée du mot index.">i</Option>
      <Option message="Ce n’est pas la bonne réponse. “i” est la variable utilisée par convention, elle est dérivée du mot index.">j</Option>
    </MultipleChoice>

    3. Comment écrit-t-on une boucle for qui a pour valeur initiale=1 et pour valeur finale=10, avec un pas de 1 ? 

    <MultipleChoice>
      <Option message="Ce n’est pas la bonne réponse. La structure d'une boucle for est 'for variable = valeur_initiale, valeur_finale do'">for i = 1 to 10 do</Option>
      <Option message="Ce n’est pas la bonne réponse. La structure d'une boucle for est 'for variable = valeur_initiale, valeur_finale do'">for i = 10, 1 do</Option>
      <Option isCorrect message="C’est la bonne réponse. La structure d'une boucle for est 'for variable = valeur_initiale, valeur_finale do'">for i = 1, 10 do</Option>
    </MultipleChoice>
