### Bonus : écrire les conditions en une seule ligne

Lorsqu'il n'y a qu'une seule commande à l'intérieur de votre condition, vous pouvez l'écrire en une seule ligne.

```lua del={1,2,3} ins={4}
if btn(➡️) then
    v.x += 1
end
if (btn(➡️)) v.x += 1
```

Vous pouvez voir que nous devons englober la condition entre parenthèses. Attention à bien respecter le nombre de parenthèses ouvrantes et fermantes !

Essayez d'appliquer ce principe pour les 4 conditions dans `_update60()`.

<details>
<summary>Voir le code complet dans `_update60()`</summary>

Voici le code pour les 4 touches :

```lua
function _update60()
  if (btn(➡️)) v.x += 1
  if (btn(⬅️)) v.x -= 1
  if (btn(⬇️)) v.y += 1
  if (btn(⬆️)) v.y -= 1
end
```
